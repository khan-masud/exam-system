ProctorX Online Exam Suite

Thank you for purchasing ProctorX. This help file walks you through installation, configuration, and troubleshooting.

Requirements

  • Node.js 20.x, npm 10.x
  • PostgreSQL 15+, Redis 7+, MinIO/S3 bucket
  • Server with HTTPS (Nginx provided)

Installation Wizard

  1. Upload the ZIP package to your server and extract.
  2. Run cp backend/.env.example backend/.env and update credentials.
  3. Navigate to /install in your browser to execute the wizard.
  4. The wizard runs migrations, seeds demo data, and creates the first admin.

License Activation

Go to /admin/settings/license, paste your Envato purchase code, and click Activate. The system will call the remote license server and store the handshake offline for 72 hours.

Payments

Configure SSLCOMMERZ, bKash, Nagad, and Rocket under Admin → Settings → Payments. IPN endpoints are automatically generated under /ipn/<provider>.
